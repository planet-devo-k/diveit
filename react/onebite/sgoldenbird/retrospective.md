# 간략 회고

자바스크립트, Node.js, 리액트 관련 내용을 강의와 핸드북을 빠르게 훑으며 정리했다. 정리하는 과정에서 그동안 인지하지 못했던 다소 모호한 개념들을 발견하고 보충할 수 있었다. 특히 자주 사용하지만 정확히 설명하기 어려웠던 부분들을 다시 확인하면서 이해를 더 명확히 할 수 있었다.

전체적으로 새로운 내용을 깊게 학습하기보다는 기존 지식을 정리하고 부족했던 부분을 보완하는 시간이었다. 짧게 훑는 방식이었지만 핵심 개념들을 다시 떠올리는 데 도움이 되었고, 앞으로 실습이나 프로젝트에서 더 자연스럽게 활용할 수 있을 것 같다.
