## Next.js Data Fetching 면접 예상 질문

1. Context API를 상태 관리 라이브러리라고 부를 수 있을까요? 리덕스나 Zustand와 비교했을 때의 결정적인 단점은 무엇인가요?

2. 리액트 외부의 스토어(Store)가 변경되었을 때, 어떻게 리액트 컴포넌트가 이를 감지하고 리렌더링을 발생시키나요? (작동 원리)

3. React 18의 동시성 렌더링(Concurrent Rendering)에서 발생할 수 있는 'Tearing(찢어짐)' 현상이란 무엇이며, 이를 해결하는 방법은 무엇인가요?

4. 최근 트렌드인 Atomic 패턴(Recoil, Jotai)과 중앙 집중식 패턴(Zustand, Redux)의 차이점과 선택 기준은 무엇인가요?

5. React Query(TanStack Query)가 등장하면서 전역 상태 관리 라이브러리의 역할은 어떻게 변화했나요?
