  # 원격(GitHub)에 새 브랜치 등록 및 푸시
  git push -u origin [새-브랜치명]

  ```

- GitHub ↔ GitBook 동기화 흐름

  ```
  [GitHub → GitBook]

  개인 브랜치에서 작업, SUMMARY.md 추가 → PR 생성 → PR 머지 → 자동으로 GitBook 에도 반영


  [GitBook → GitHub]

  GitBook 개인 Space에서 작업 → CR 제목 입력 → CR 머지 → GitHub 개인 브랜치에 반영 → PR 생성 → PR 머지
  ```
